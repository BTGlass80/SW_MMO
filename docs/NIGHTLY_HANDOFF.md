## Rules Implemented In Prototype

- Ranged difficulty by range band.
- Cover dice and full-cover blocking.

- Live ground range pressure: active remotes evaluate deterministic incoming-fire windows on a short clock, with target-specific cadence/phase, peeking/tucked, flanking/repositioning, reload/weapon-cycle, covering-fire, morale-hesitation, and fallback metadata staggering which remotes fire on each automatic tick; `V` remains an immediate debug/manual volley from every eligible armed remote.
- Range target return-fire eligibility separates shootable targets from armed combatants, so inert scale/armor drills can be shot without adding bogus remote fire.
- Range target live-fire cadence keeps automatic encounter pressure paced without changing the WEG D6 attack, defense, damage, or soak resolution path.
- Range target suppression metadata lets successful player hits push active remotes off upcoming live-fire ticks without changing WEG attack or damage resolution.
- Range target pinning metadata lets close player misses briefly push configured remotes off upcoming live-fire ticks through a smoke-tested model helper, adding suppressive pressure behavior without changing WEG hit, damage, or soak results.
- Range target peeking metadata lets covered remotes alternate exposed and tucked live ticks, delaying shots as encounter behavior without changing WEG attack, cover, damage, or soak rules.
- Range target flanking metadata lets selected remotes spend configured live ticks repositioning instead of firing, adding NPC pressure variation without changing WEG attack, cover, damage, or soak rules.
- Range target reload metadata lets selected remotes spend configured live ticks cycling weapons instead of firing, adding automatic-fire pacing without changing WEG attack, damage, or soak rules.
- Range target covering-fire metadata lets selected remotes spend configured live ticks applying non-damaging pressure instead of resolving a shot, making live pressure less metronomic without changing WEG attack, damage, or soak rules.
